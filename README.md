# PE Informer

Легкая автономная утилита для анализа Portable Executable (PE) файлов, написанная на C++ и ImGui. Создана для малварь-аналитиков и реверс-инженеров для быстрого изучения структуры бинарников, вычисления энтропии и эвристического детектирования упаковщиков.

### Особенности
* **Парсинг PE-заголовков:** Чтение DOS, NT, File и Optional заголовков (с поддержкой архитектур x86 и x64).
* **Расшифровка Rich Header:** Поиск XOR-ключа и декрипт недокументированного заголовка MSVC для точной идентификации версий компилятора, линкера и среды сборки.
* **Определение тулчейна и подсистем:** Распознавание компиляторов (MSVC, MinGW, Delphi) и типа приложения (GUI, Console, SYS, DLL).
* **Эвристический детект упаковщиков:** Выявление пакеров, протекторов и инсталляторов (UPX, VMProtect, Themida, NSIS и др.) на основе названий секций и структурных аномалий (например, отсутствие секции `.text` или высокая энтропия).
* **Энтропия Шеннона (Shannon Entropy):** Математический расчет энтропии для всего файла и каждой отдельной секции для выявления сжатых или зашифрованных данных.
* **Современный UI:** Написан на базе ImGui + DirectX 11. Поддержка нативной темной темы Windows (DWM Immersive Dark Mode), минималистичный дизайн без лишних рамок и поддержка Drag & Drop.

### Стек технологий
`C++` `WinAPI` `ImGui` `DirectX 11` `PE Format`

### Использование
Просто запустите утилиту и перетащите любой `.exe`, `.dll` или `.sys` файл в окно программы (Drag & Drop), либо нажмите на иконку папки для выбора файла через проводник.
